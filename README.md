# Delivery-Project-PHP-Laravel
This is my Final Web Server Delivery Project using PHP and Laravel. Integrated API ExchangeRate from CAD to USD for delivery rate charge. Aditionally, used mySQL DB to store orders. User can manage their orders and deliveries, they can ship them through 3 delivery providers (Internal, Canada Post or UPS). They can track their shipment status as well. After user puts in all information needed for a delivery to be ordered, their shipping rate is going to be converted from CAD to USD using ExchangeRate API which are all logged.
